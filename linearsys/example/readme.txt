Building the example:

To build the example, just type:

Make

And the executable will be placed inside the bin folder.
To execute the program you must use the run.bat or run.sh script depending on whether if you are in a 
Microsoft command line or in a Linux shell.
Under Win command line
change to bin folder and type:

Run

Under Linux or under the MinGW shell
change to bin folder and type:

./run.sh

The result of testing will be placed in the out.txt file inside the bin folder.
